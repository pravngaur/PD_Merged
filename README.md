# Page Designer Reference Components

This repo is to help Developers & Architects to provide some inspiration to create their own custom Components & Pages.

## What's Covered
Implementation of the custom YouTube & Instagram components is provided. Once deployed these components fetch the content from respective sites; it requires you to configure the components config in BM.

## What's Next?
Once you have understood the design and implementation pattern of these components, you are good to create your own awesome components & pages.
